# Postman Collection & Environment

## Files
- [athleticaos_local_environment.json](environments/athleticaos_local_environment.json)
- [athleticaos_postman_full.json](full/athleticaos_postman_full.json)

## Usage
1. Import both files into Postman.
2. Select "AthleticaOS Local" environment.
3. Run "Auth/Login" to get a token.
4. The token is automatically saved to the environment variable `token`.

## Notes
- Endpoints for Calendar, File Uploads, and Utility are placeholders as they were not found in the current backend codebase.
